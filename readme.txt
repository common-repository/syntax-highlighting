HighLight Code allows you to add syntax highlighting to the snippets contained in your articles.
NOTE (beta): This plugin is very felt, it is still in development phase!

== Description ==
The plugin is based on the javascript library HighLight Js, it offers the possibility to choose among 70 styles of coloring. You can choose a default color in the back office. You can also add a widget to let the choice of coloring to visitors.

= Features Include =

* Add syntax highlighting to your code snippets.
* Choose style from more than 50 choices.
* A widget allows visitors to make their choice on the front

== Configuration ==

1. Activate the plugin through the "Plugins" menu in WordPress.
2. Go to the plugin setting page and add the types of articles or add syntax highlighting.
3. Then choose a theme for the coloring, demo here :[HighLight JS](https://highlightjs.org/static/demo/).
4. To let visitors change the coloring style, go to your dashboard to add the widget to the sidebar of your choice.
5. Finally, when you add a code in a publication, do not forget to check ON in the Syntax Highlighting metabox.
